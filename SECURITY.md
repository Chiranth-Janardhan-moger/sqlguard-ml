# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 1.1.x   | Yes       |
| 1.0.x   | Yes       |
| 0.1.x   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability within SQLGuard ML, please do not disclose it publicly. Instead, submit an issue with the label `security` or contact the maintainer directly.
